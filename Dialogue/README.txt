RULES FOR DIALOGUE
#1 each dialogue must be preceeded by a 6-digit ID code: made up of 3 two digit codes
#2 all dialoge must be grouped with their ID on their own, singular, line.
#3 The Dialoge ID will follow this format: (Room ID)-(Character ID)-(dialogue number)
#4 (Dialoge ID) 00-00-00 is reserved for Error text

#5 each interaction has an assigned dialoge offset
#5.1 Take is assigned the offset of 0.
#5.2 talk is assigned the offset of 1
#5.3 Examine is assigned the offset of 2
#5.4 USE is assigned the offset of 3
#5.5 Traverse is assigned the offset of 4
