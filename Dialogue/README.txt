RULES FOR DIALOGUE
#1 each room gets it's own file, which will be loaded when you enter the room
#2 each dialogue must be preceeded by a 6-digit ID code: made up of 3 two digit codes
#3 The Dialoge ID will follow this format: (Room ID)-(Character ID)-(dialogue number). Yes this means that every dialogue ID in a file will have the same first two digits
#4 (Character ID) 00 is resserved simple inanimate objects, with only a single line of dialogue. example: a pot that you click on and the dialogue says: "It's a pot"
#5 (Dialoge ID) 00-00-00 is reserved for default text, to be displayed whenever the program is uncertain what text to display
